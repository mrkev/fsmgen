Fsmgen

This site lets you design FSMs using a DSL. Here's how to use it;

Define a state:

state q0

Connect the state to other states:

state q0

state q1 =
  | back -> q0

Define an initial state, and some final states:

initial state q0 =
  | stay -> q0
  | move -> q1
  | jump -> q2

state q1 =
  | back -> q0
  | stay -> q1
  | move -> q2

final state q2 =
  | stay -> q2
  | restart -> q0
